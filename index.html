FRC 6045 Programming Website!

The purpose of this website is to make it easier to find programming-related things that are used or made by FRC 6045.

Applications

PathPlanner is the software that 6045 uses to make autos.
https://github.com/mjansen4857/pathplanner/releases
Click on the large heading with the green "Latest" symbol next to it.
Scroll to the bottom and click on a file that matches your operating system.
For school-provided-MacBook users, install the ZIP file and then enter "xattr -d com.apple.quarantine file_path" in the terminal. You can quickly get the file path by dragging the installer into the terminal.

WPILib Visual Studio Code is the software 6045 uses to edit and deploy code.
https://docs.wpilib.org/en/stable/docs/zero-to-robot/step-2/wpilib-setup.html
For school-provided-MacBook users, you must first install JDK and Xcode, both of which should be available on Self Service. You will need the administrator credentials to download XCode.

GitHub Desktop is the software that 6045 uses to access code across multiple computers.
https://desktop.github.com/download

REV Hardware Client is the software that 6045 uses to assign CAN IDs to motors. This is only available for Windows.
https://docs.revrobotics.com/rev-hardware-client/gs/install

National Instruments' FRC Game Tools includes the Driver Station, software that is necessary to control the robot. This is only available for Windows.
https://www.ni.com/en/support/downloads/drivers/download.frc-game-tools.html
